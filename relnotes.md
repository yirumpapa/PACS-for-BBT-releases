**PACS for BBT v1.1.0** (2026-07-03)

**New**
- Airway tracing: right-click to place dots, then connect them into a smooth 3D curve (Connect Dots)
- Perpendicular cross-section viewer: CT sections perpendicular to the traced airway, with Sync to the main views, cutting-plane display, real-time orientation labels, and flip/rotate
- Redesigned toolbar icons; double-click a worklist row to switch study and close the window
- Updated user manual (English / Korean)

**File**: `PACS-for-BBT.exe` (92 MB, Windows 10/11 64-bit, no installation)
